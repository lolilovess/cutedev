## üõ†Ô∏è 3. HOW TO APPLY THE HIJACK
it's a simple stitch:
1. **Open github.dev**: Navigate to any repository on GitHub and press the `.` key on your keyboard to open `github.dev`.
2. **Open Settings JSON**: Hit `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) to open the Command Palette. Type "Open User Settings (JSON)" and select it.
3. **Paste the Snippet**: Copy the entire content of `settings.json` from this repository and paste it into your `settings.json` file, replacing any existing customizations if you want the full experience.
4. **Comic Sans Check**: Ensure your operating system has "Comic Sans MS" installed for the font to render correctly.
5. **Reload & Relax**: Reload the page or the editor. Your workspace is now a cozy crafting haven.
